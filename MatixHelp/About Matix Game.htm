About Matix Game

The Matix Game is a board game with 8 rows and 8 column. The porpose of the game is to collect hier score than the other player when the game ended. The board is generated randomly by the server with values between -15 to 15. One of the players play horizontaly and the other play verticaly. The server defines the players direction and in case of human players define who is the first one. While playing with server the human player will start.
